### Hướng Dẫn Cài Đặt ClawDesktop trên VPS (Ubuntu 20.04/22.04)

**Mô tả**: ClawDesktop là một ứng dụng giả lập môi trường người dùng (như Cloud PC) dùng để treo game, chạy automation hoặc các phần mềm yêu cầu màn hình desktop thực.

#### 1. Yêu cầu hệ thống (System Requirements)
- OS: Ubuntu 20.04 hoặc 22.04 LTS (Khuyên dùng Ubuntu 20.04 để độ ổn định cao nhất).
- RAM: Tối thiểu 4GB (Khuyến nghị 8GB nếu chạy nhiều tác vụ nặng).
- CPU: Tối thiểu 2 Cores.
- Disk: 30GB trống.

#### 2. Các bước cài đặt cơ bản (Installation Steps)
1. Truy cập SSH vào VPS bằng quyền `root`.
2. Cập nhật hệ thống:
   `apt update && apt upgrade -y`
3. Cài đặt các gói phụ thuộc cần thiết (curl, git, wget):
   `apt install curl git wget unzip -y`
4. Cài đặt môi trường Xfce4 (Giao diện Desktop nhẹ) và XRDP (Remote Desktop Protocol):
   `apt install xfce4 xfce4-goodies xrdp -y`
5. Cấu hình XRDP sử dụng Xfce4:
   `echo xfce4-session > ~/.xsession`
   `systemctl restart xrdp`
   `systemctl enable xrdp`
6. Mở port cho phép kết nối Remote Desktop:
   `ufw allow 3389/tcp`

#### 3. Chạy App ClawDesktop
- Chạy lệnh khởi tạo môi trường container cho ClawDesktop:
  `bash <(curl -s https://raw.githubusercontent.com/clawdesktop/install/main/setup.sh)`
- Sau khi chạy xong script, màn hình sẽ hiển thị `User` và `Password` đăng nhập.
- Mở máy tính cá nhân của bạn, mở chương trình "Remote Desktop Connection" (trên Windows) hoặc "Microsoft Remote Desktop" (trên Mac).
- Nhập IP của VPS và đăng nhập bằng thông tin cung cấp ở trên.

#### 4. Các lỗi thường gặp (Troubleshooting / FAQ)
**Hỏi**: Tôi kết nối Remote Desktop báo màn hình đen (Black Screen) / văng ngay lập tức?
**Đáp**: Lỗi này do XRDP không nhận được xsession hoặc đụng độ session. Bạn cần đổi port hoặc chạy lệnh fix: `sed -i.bak '/fi/a #xrdp multiple users configuration \\n xfce-session \\n' /etc/xrdp/startwm.sh` rồi gõ `systemctl restart xrdp`.

**Hỏi**: Có thể chạy tool auto (macro chuột phím) trên môi trường này không?
**Đáp**: Hoàn toàn có thể. ClawDesktop mặc định đã config giả lập display đầy đủ để các tool như auto click, pygui, selenium chạy mượt mà không bị lỗi "No display found".

**Hỏi**: Làm sao để đổi độ phân giải màn hình VPS?
**Đáp**: Mặc định ClawDesktop sẽ auto-resize theo độ phân giải màn hình máy khách (máy bạn đang dùng để Remote). Bạn chỉ cần tắt Remote Desktop, chỉnh lại setting màn hình trong app RDP trước khi connect là VPS sẽ tự lấy theo.

**Hỏi**: Có cài được Google Chrome không?
**Đáp**: Có, truy cập Terminal trên VPS gõ lệnh:
`wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb`
`apt install ./google-chrome-stable_current_amd64.deb -y`
